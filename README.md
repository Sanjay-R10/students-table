# 🎓 Students Table

A full-stack inspired frontend application built with React.js for managing student records with full CRUD operations.

## 🔗 Live Demo
[https://students-table-lime.vercel.app](https://students-table-lime.vercel.app)

## 🛠️ Tech Stack
- React.js (Vite)
- Tailwind CSS (CDN)
- SheetJS (xlsx) — Excel download
- Lucide React — Icons
- Vercel — Deployment

## ✅ Features
- View all students in a clean table
- Add new student with form validation
- Edit existing student with pre-filled form
- Delete student with confirmation dialog
- Search/filter students by name or email
- Download filtered or full data as Excel (.xlsx)
- Simulated loading state on initial render
